date: '2026-07-02'
description: Out with Vue 2 and Vuetify, in with a text-first React site.

The old version of this site was a Vue 2 + Vuetify app I built for fun years
ago. It served its purpose, but it was supremely outdated and hard to add to —
so writing anything here never happened.

This rebuild is Next.js with MDX. Adding a post is now just dropping a
markdown file into a folder, which removes my last excuse for not writing.

The old site lives on in the `vue-legacy` branch of
[the repo](https://github.com/izman48/iesawazeer.com).
