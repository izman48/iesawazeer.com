# IesaWazeer.com

This is a personal portfolio website made using VueJS. I have used Vuetify and the backend is handled on firebase.

## Installation
Install npm on your pc

In the frontend folder to download the dependencies:
```
npm install
```

## Usage

To run dev server:

```
npm run serve
```
